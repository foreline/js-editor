# Block type selector removal

This document describes the decision to remove the block type selector from the editor's. The block type selector was a dropdown menu that allowed users to change the type of a block (e.g., from paragraph to heading) after it was created. It should be triggered by typing the `/` character at the beginning of a block. But anyway the functionality was not implemented and the dropdown was not shown in the UI.

We are not sure if we need this feature at all, and if we do, we can implement it later in a more user-friendly way.

Now it only polutes the editor with the placeholder text "Type / to choose a block type".

## Decision
Remove the placeholder text "Type / to choose a block type", leaving the corresponding code (if any present) for future implementation if needed.
