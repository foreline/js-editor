# Empty Block Arrow Navigation Issue

This document outlines an issue related to arrow key navigation in empty blocks within the editor. The goal is to identify the root cause of the problem and propose a solution to ensure smooth navigation for users when working with empty blocks.

## Current behavior
When a user creates an empty block (e.g., by pressing Enter at the end of a block), they are unable to navigate (put a cursor) into that empty block using the arrow keys. This can lead to a frustrating user experience, as users may expect to be able to move the cursor into the new block immediately after creating it. So they have to click into the block with the mouse to start typing, which is not ideal.
