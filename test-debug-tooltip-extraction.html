Debug Tooltip Extraction Test

This test verifies that the debug tooltip functionality has been successfully extracted to its own class.

Test Controls:

Debug Mode: DISABLED

How to Test:

  1. Click "Toggle Debug Mode" to enable debug tooltips
  2. Click on different blocks in the editor to see debug tooltips
  3. Try typing content and see tooltips update
  4. Test scroll and resize events to see tooltip repositioning
  5. Toggle debug mode off to ensure tooltips are properly removed

Expected Behavior:

  • Debug tooltips should only appear when debug mode is enabled
  • Only one tooltip should be visible at a time (for the active block)
  • Tooltips should reposition automatically when scrolling or resizing
  • All tooltips should be removed when debug mode is disabled
  • The Editor class should be smaller and cleaner without debug tooltip code
